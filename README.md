# Student Participation and Activity Heatmap System

Backend application to track and analyze student participation and activity data using heatmap logic.

## Tech Stack

* Node.js
* Express.js
* MySQL
* JavaScript
* RESTful API

## Features

* Track student participation and activity records
* Generate heatmap data based on activity patterns
* REST APIs for data collection and retrieval
* CRUD operations for managing student data
* Structured database storage using MySQL

## API Endpoints

* GET /activities → Retrieve activity data
* POST /activities → Add new activity record
* PUT /activities/{id} → Update activity
* DELETE /activities/{id} → Delete activity

## Project Structure

* Routes → API endpoints
* Controllers → Request handling
* Config → Database configuration
* Models → Data structure

## How to Run

1. Clone the repository
2. Install dependencies using npm install
3. Configure MySQL database in config file
4. Run server using node app.js or npm start
5. Test APIs using Postman
